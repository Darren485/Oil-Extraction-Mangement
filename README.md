# Oil Extraction Management System

## Overview
A Java console-based application to manage oil rigs, extraction records, maintenance records, and maintenance crews using Hibernate and PostgreSQL.

## Features
- Manage oil rigs
- Track extraction records
- Log maintenance records
- Register and manage maintenance crews
- View most active crews and update crew status

## Technologies
- Java
- Hibernate ORM
- PostgreSQL
- Maven
